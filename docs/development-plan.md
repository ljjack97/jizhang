# 开发执行计划

## 开发原则
- **小步推进**：每个 Phase 完成并验证后再进入下一个
- **可运行**：每个 Phase 结束时代码必须可运行、无报错
- **先骨架后血肉**：先搭框架，再加功能，最后做美化
