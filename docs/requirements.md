# 开发需求文档

## 项目名称
记账软件（JiZhang）

## 项目目标
开发一款运行在手机上的个人记账 PWA 应用，支持手动录入支付记录，以时间排序展示所有支付记录，提供搜索、统计图表和智能记账功能。

## 目标用户
个人用户，需要追踪和管理日常支付记录。

## 功能需求

### FR-01 支付记录列表（首页）
- 所有记录按支付时间倒序排列（最新在上）
- 每条记录卡片展示：金额、收支类型（收入绿/支出红）、分类标签、商家、支付时间
- 支持左滑删除（van-swipe-cell + 红色删除按钮）
- 空数据时显示引导提示（EmptyState 组件）
- 点击卡片进入详情页
- 首次打开显示欢迎弹窗（操作手册 + 功能列表）
- 摇一摇唤起记账（DeviceMotion API，阈值 15m/s²，冷却 1.5s）

### FR-02 记录详情
- 展示完整字段：金额、类型、分类、商家、支付时间、订单编号、支付平台、备注
- 编辑模式：可修改全部字段，保存前校验
- 编辑离开确认：未保存修改时弹出确认弹窗
- 提供删除按钮，删除后返回首页

### FR-03 手动添加记录
- 表单字段：
  - 金额（必填，数字输入，保留两位小数）
  - 收支类型（收入/支出切换）
  - 消费分类（8 项：餐饮/交通/购物/住宿/娱乐/转账/红包/其他）
  - 商家（可选，文本输入）
  - 支付时间（必填，默认当前时间，日期选择器）
  - 订单编号（可选，文本输入）
  - 支付平台（必填，下拉选择：微信/支付宝/银行卡/其他）
  - 备注（可选，文本输入）
- 支持快速记账（?auto=1 预填模式）
- 提交前表单校验
- 提交成功后跳转首页 + 音效/震动反馈

### FR-04 搜索与筛选
- 关键词搜索：匹配订单编号、备注、金额、商家
- 日期范围筛选：选择起止日期
- 分类筛选栏：横向滚动，7 大类 + "全部"按钮
- 搜索结果按时间倒序排列

### FR-05 智能记账
- 剪贴板识别：读取系统剪贴板 → 解析金额/商家/平台 → 弹窗确认 → 一键记账
- CSV 账单导入：微信/支付宝账单格式解析 → 自动分类匹配（100+ 关键词） → 预览 → 批量导入
- 自动记账弹窗（AutoAddPopup）：展示预填信息 + 分类选择 + 一键确认

### FR-06 统计图表
- 收支摘要卡片（收入/支出金额 + 笔数）
- SVG 环形饼图：支出分类占比，扇区可点击高亮，图例联动
- SVG 柱状趋势图：每日收支双柱对比
- 周期切换：今日 / 本月 / 本年
- 统计弹窗（首页）：蓝底总额卡片 + 收支明细表

### FR-07 个人中心
- 头像：点击放大预览 + 从相册选择上传（canvas 缩放至 200×200 JPEG）
- 昵称：弹窗修改，20 字限制
- 手机号：弹窗修改，11 位校验，自动脱敏显示
- 退出登录：确认弹窗

### FR-08 设置页
- 添加按钮偏好：悬浮按钮（FAB）/ 顶部导航栏
- 交互开关：摇一摇 / 音效 / 震动（独立控制）
- 清除全部记录（确认弹窗 + 防二次填充）
- CSV 账单导入入口

### FR-09 数据持久化
- Pinia + pinia-plugin-persistedstate 自动同步 localStorage
- 刷新页面或关闭浏览器后数据不丢失
- 3 个持久化 key：jizhang-records / jizhang-preferences / jizhang-user

### FR-10 PWA 支持
- 可安装到手机桌面
- 支持离线访问
- 桌面快捷方式：添加支出 / 添加收入 / 查看统计
- 主题色 #5B9BD5

### FR-11 交互体验
- 老式收音机音效（Web Audio API 合成 5 种音效）
- 震动反馈（navigator.vibrate，4 种模式）
- 摇一摇唤起记账（可关闭）
- 双击提交保护（3 处：编辑保存/自动记账/CSV 导入）

### FR-12 可访问性
- WCAG AA 对比度（全站文字色 ≥ 4.5:1）
- 15+ aria-label 覆盖图标按钮、SVG 图表
- 键盘导航（tabindex + @keydown.enter/space）
- SVG 图表角色标注（role="img" + aria-label）

## 非功能需求

### NFR-01 移动端适配
- 以手机屏幕为主要目标，宽度 375-414px 为最佳体验区间
- 在平板和桌面端也能正常显示（居中窄版布局，max-width: 480px）

### NFR-02 性能
- 首次加载时间 < 2 秒
- PWA Service Worker 预缓存 32 个资源
- 头像图片缩放至 ≤ 30KB，不撑爆 localStorage

### NFR-03 可用性
- 界面简洁直观，首次访问显示欢迎引导
- 表单错误提示清晰
- 操作有即时反馈（加载状态、成功/失败 toast、音效、震动）
