# 设计规范

## 设计原则
- **简洁**：减少视觉噪音，每个页面只聚焦一个核心操作
- **直观**：符合移动端操作习惯，按钮位置方便拇指触达
- **高效**：减少操作步骤，常用功能一键可达

## 色彩系统

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | `#5B9BD5` | 淡蓝色，用于导航栏、按钮、强调元素 |
| 主色浅 | `#E8F4FD` | 极淡蓝，用于卡片背景、选中状态 |
| 主色深 | `#3A7CC3` | 深蓝，用于按钮按下态 |
| 成功/收入 | `#52C41A` | 绿色 |
| 警告 | `#FAAD14` | 橙色 |
| 危险/删除 | `#FF4D4F` | 红色 |
| 文字主色 | `#333333` | 深灰，标题和正文 |
| 文字辅色 | `#999999` | 浅灰，次要信息 |
| 背景色 | `#F5F7FA` | 页面背景 |
| 卡片白 | `#FFFFFF` | 卡片和内容区背景 |
| 分割线 | `#EBEDF0` | 列表分隔 |

## 组件设计

### 记录卡片 (RecordCard)
```
┌─────────────────────────────────┐
│  ¥ 128.50          微信支付     │  ← 金额 + 平台标签
│  2024-01-15 14:30              │  ← 支付时间
│  订单: 20240115...             │  ← 订单编号（截断）
│                        详情 >  │  ← 进入详情
└─────────────────────────────────┘
```

### 顶部导航 (AppHeader)
```
┌─────────────────────────────────┐
│  ← 返回        标题         🔍  │  ← 左侧返回 + 居中标题 + 右侧搜索
└─────────────────────────────────┘
```

### 空状态 (EmptyState)
```
         ┌─────┐
         │ 📋  │                    ← 图标
         └─────┘
    还没有支付记录                   ← 提示文字
  点击下方按钮添加第一笔             ← 引导文字
```

## 移动端适配
- 最大内容宽度：`480px`，超宽屏幕居中显示
- 触摸目标最小尺寸：`44x44px`
- 底部导航高度：`50px`（使用 Vant Tabbar）
- 安全区域：适配 iPhone 刘海屏（`safe-area-inset-*`）
