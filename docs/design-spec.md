# 设计规范

## 设计原则
- **简洁**：减少视觉噪音，每个页面只聚焦一个核心操作
- **直观**：符合移动端操作习惯，按钮位置方便拇指触达
- **高效**：减少操作步骤，常用功能一键可达

## 组件设计

### 记录卡片 (RecordCard)
```
┌──────────────────────────────────────┐
│  -¥ 128.50                    餐饮   │  ← 金额（红/绿） + 分类标签
│  麦当劳                     14:30    │  ← 商家 + 支付时间
│                           滑动删除 →│  ← 左滑露出红色删除按钮
└──────────────────────────────────────┘
```

### 添加按钮
- **FAB 模式**（默认）：右下角悬浮圆形蓝色按钮（+）
- **TopBar 模式**：顶部导航栏右侧图标按钮
- 可在设置页切换

## 移动端适配
- 最大内容宽度：`480px`，超宽屏幕居中显示
- 触摸目标最小尺寸：`44x44px`
- 底部导航高度：`50px`（使用 Vant Tabbar）
- FAB 按钮距右下角：`24px`
- 安全区域：适配 iPhone 刘海屏（`safe-area-inset-*`）

## 可访问性
- 文字对比度 ≥ 4.5:1（WCAG AA 标准）
- 交互元素提供 aria-label + role + tabindex
- SVG 图表添加 role="img" + aria-label
- 键盘导航支持（Enter/Space 激活）
- 空状态装饰图标标记 aria-hidden="true"
