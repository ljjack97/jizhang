#### 功能说明

**剪贴板识别链路**：
1. Home.vue `onMounted` → 调用 `readClipboard()` 读取系统剪贴板
2. `isPaymentText()` 判断是否为支付通知文本
3. `parsePaymentText()` 提取金额（¥/￥/元/纯数字）、商家（关键词匹配）、平台（微信/支付宝/银行）
4. 识别成功 → 弹出 AutoAddPopup，展示预填信息
5. 用户选分类 → "立即记账" 一键保存
6. "需要更多信息" → 跳转 Add.vue?auto=1，完整表单预填

**CSV 导入链路**：
1. 设置页点击"导入 CSV 账单" → 系统文件选择器
2. 读取文件 → `parseCSVBill()` 自动检测来源（微信/支付宝/通用）
3. 列名智能匹配（兼容多别名） → 逐行解析
4. `guessCategory()` 根据商家关键词自动分类（100+ 关键词覆盖）
5. 预览弹窗：收支摘要 + 前5条预览 + 分类分布 + 解析警告
6. 确认导入 → 批量写入 Store + localStorage

**商家→分类映射**（部分）：
- 餐饮：麦当劳/肯德基/星巴克/美团/外卖/火锅/...
- 交通：滴滴/曹操/地铁/公交/高铁/加油/...
- 购物：淘宝/天猫/京东/拼多多/商场/...
- 住宿：酒店/宾馆/民宿/如家/汉庭/...
- 娱乐：电影/KTV/游戏/充值/健身/...
- 转账：转账/红包/汇款/工资/提现/...

#### 音效设计

| 音效 | 触发场景 | 合成方式 |
|------|---------|---------|
| `success` | 添加记录成功 | 740→880Hz 双频正弦波 + 倍频谐波 |
| `tap` | 按钮点击 | 220→80Hz 三角波短脉冲 |
| `shake` | 摇一摇检测到 | 带通滤波白噪音 + 180→260Hz 正弦波 |
| `delete` | 删除记录 | 330→110Hz 三角波下行 |
| `toggle` | 开关切换 | 600→900Hz 锯齿波上扫 |

#### 震动模式

| 模式 | 模式数组 | 使用场景 |
|------|---------|---------|
| short | `10ms` | 按钮点击 |
| success | `[30, 60, 30]` | 操作完成 |
| warning | `[40, 50, 40, 50, 40]` | 删除确认 |
| shake | `50ms` | 摇动检测反馈 |
